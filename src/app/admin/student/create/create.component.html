Formulario de Registro de Estudiante

Panel de Administración

Bienvenido al sistema de registro de usuarios. Como administrador, puedes agregar nuevos estudiantes a la plataforma completando el formulario de registro. Asegúrate de ingresar información válida.

Guía rápida:

  • Completa todos los campos obligatorios.
  • Verifica que el número de documento y el teléfono sean correctos.
  • Si tienes dudas, consulta el manual del administrador.
